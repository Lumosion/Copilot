# 周边美食发现 - 美食记录与推荐平台

一个帮助你记录、分享和发现周边美食的Web应用。支持添加美食店铺信息、价格评估、星级评分、距离计算，以及通过转盘随机决定今天吃什么。

## 功能特点

- 🍜 **美食管理** - 记录美食名称、店铺、价格、评分等完整信息
- ⭐ **评分系统** - 支持1-5星评分，快速筛选高分美食
- 📍 **距离计算** - 显示店铺距离和预估步行时间
- 🎡 **转盘抽选** - 随机转盘帮你决定今天吃什么
- 💾 **本地存储** - 数据保存在浏览器，下次访问自动加载
- 📱 **响应式设计** - 完美支持手机、平板和电脑访问
- 🎨 **美观界面** - 现代化设计，渐变色彩，流畅动画

## 页面内容

- **首页（Hero）** - 精美的欢迎页面，快速导航到各功能
- **美食列表** - 展示所有美食，支持按星级筛选
- **添加美食** - 表单输入新美食信息（名称、店铺、价格、评分、距离、类型）
- **转盘决定** - 互动式转盘，随机选择一家美食店

1. 克隆此仓库：
```bash

cd Copilot
```

2. 使用任意 HTTP 服务器预览：
```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx http-server

# 或直接用浏览器打开 index.html
```

3. 在浏览器中访问 `http://localhost:8080`

## 使用说明

### 添加美食
1. 点击导航栏的"添加美食"或首页的"添加美食"按钮
2. 填写表单信息：
   - **美食名称**：例如"川味火锅"
   - **店铺名称**：例如"老四川火锅店"
   - **人均价格**：输入数字，例如80元
   - **星级评分**：选择1-5星
   - **距离**：输入距离（米），例如500米
   - **类型**：选择美食类型（中餐、西餐、日料等）
3. 点击"添加美食"提交

### 浏览美食
- 在"美食列表"页面查看所有已添加的美食
- 使用星级筛选按钮快速找到高分美食
- 每张卡片显示：美食emoji、名称、店铺、价格、评分、距离、步行时间

### 转盘决定
1. 点击"今天吃什么"进入转盘页面
2. 点击"开始转动"按钮
3. 转盘旋转后会随机停在一家美食店
4. 结果会显示选中的美食及详细信息

## 数据说明

- 应用自带6家示例美食数据
- 所有数据保存在浏览器的 localStorage 中
- 清除浏览器数据会重置为初始状态
- 数据仅保存在本地，不会上传到服务器

- **Vanilla JavaScript** - 原生JS实现所有功能
