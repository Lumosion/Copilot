# 明日方舟 Arknights - 官方网站

完全仿照明日方舟官方网站（https://ak.hypergryph.com/）制作的演示页面，包含完整的数据和功能。

## 功能特点

- 🎮 **官方风格** - 完全复刻官方网站的设计风格和布局
- 🌟 **完整数据** - 包含真实的游戏数据、干员信息、活动资讯等
- 📱 **响应式设计** - 完美支持移动端和桌面端访问
- 🎯 **丰富内容** - 涵盖最新情报、世界观、干员展示、游戏特色等板块
- 💫 **精美动画** - 流畅的滚动动画、悬停效果和粒子背景
- 🎨 **专业设计** - 深色主题、渐变色彩、现代化UI设计

## 页面内容

- **首页（Hero）** - 大气的游戏标题展示，带有动态粒子效果
- **最新情报** - 游戏活动、更新公告、联动信息等
- **世界观** - 泰拉世界设定、源石、罗德岛等背景介绍
- **干员展示** - 精英干员大图展示和干员列表
- **游戏特色** - 策略塔防、干员收集、剧情、美术等特色介绍
- **游戏资料** - 游戏截图、宣传视频、壁纸下载等
- **下载专区** - iOS、Android、哔哩哔哩、PC模拟器等多平台下载

## 在线访问

访问 GitHub Pages 查看效果：`https://lumosion.github.io/Copilot/`

## 本地预览

1. 克隆此仓库：
```bash
git clone https://github.com/Lumosion/Copilot.git
cd Copilot
```

2. 使用任意 HTTP 服务器预览：
```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx http-server

# 或直接用浏览器打开 index.html
```

3. 在浏览器中访问 `http://localhost:8080`

- **下载渠道** - App Store、Google Play、哔哩哔哩、PC模拟器
